# FinanceFocus Backend

## Proje Tanımı

FinanceFocus, kullanıcıların kişisel finanslarını tek bir platform üzerinden yönetebildiği modern bir Full Stack finans yönetim sistemidir.

Bu proje yalnızca bir CRUD uygulaması değildir. Amaç; gerçek hayatta kullanılabilecek, ölçeklenebilir, güvenli ve sürdürülebilir bir backend mimarisi geliştirmektir.

Frontend tarafı React + TypeScript ile tamamlanmıştır. Bu repository içerisinde geliştirilecek backend, mevcut frontend ile tam uyumlu çalışacak şekilde tasarlanacaktır.

---

# Projenin Amacı

Bu backend'in temel amacı;

- Güvenli kullanıcı yönetimi sağlamak
- Finansal verileri merkezi olarak yönetmek
- Frontend ile REST API üzerinden haberleşmek
- İş kurallarını backend tarafında yürütmek
- Gerçek veriler üzerinde analiz ve raporlama yapmak
- AI Financial Assistant modülünü desteklemek
- Production seviyesine yakın bir mimari oluşturmak

# AI Financial Assistant

Forecast Engine ile AI Assistant farklı yapılardır.

Forecast Engine;

- finansal veriler üzerinde matematiksel analiz yapar.

AI Financial Assistant ise;

- OpenRouter üzerinden çalışacak,
- kullanıcının finansal verilerini yorumlayacak,
- öneriler sunacak,
- soru-cevap gerçekleştirecek,
- backend üzerinden güvenli şekilde yönetilecektir.

LLM API anahtarları hiçbir zaman frontend tarafında tutulmayacaktır.
